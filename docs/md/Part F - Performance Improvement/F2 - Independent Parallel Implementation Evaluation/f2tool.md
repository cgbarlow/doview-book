# DoView Tool F2 — Independent Parallel Implementation Evaluation

> **Pair:** [Question](f2question.md) · Tool (this page)

A purchaser/funder interested in optimizing an initiative's implementation can employ an independent organization to perform an ongoing independent implementation evaluation (formative evaluation) of the initiative.

## Diagram

```mermaid
flowchart TD
    funder["Government agency or funder"]
    init["Initiative"]
    eval["Independent Implementation Evaluators"]
    init -- "Normal reporting" --> funder
    eval -- "Implementation evaluators can send a copy of their regular formal implementation evaluation report to the funder" --> funder
    eval -- "Implementation evaluators interact with the initiative informally and make suggestions for improving the initiative's implementation" --> init
    eval -- "Implementation evaluators formally report regularly to the initiative governance regarding issues that need to be addressed (e.g. every 3 months)" --> init
```
