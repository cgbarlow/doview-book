# How can an organization, agency, provider, policy or initiative's performance be improved?

> **Pair:** Question (this page) · [Tool](f01tool.md)

Continuously improving the performance of organizations, agencies, providers, policies, and initiatives is a key aspect of ensuring that they will achieve their outcomes. In the government context, this is Step 12 in the Government Planning and Implementation Cycle.

Traditionally, this is done either by the initiative itself or by bringing in a team to review the organization, provider, policy or initiative. This can be a quality assurance team of some sort or an independent team of implementation evaluators (as is used in Step F - Implementation Evaluation for Optimization - from the DoView Planning Framework (D1). Such performance reviews or implementation evaluation activity looks at both the initiative's capability and capacity, and whether these are translating into achieving its priorities and outcomes. When using the DoView Planning approach, an efficient way of doing such performance improvement work is to do it against the initiative's DoView strategy/outcomes diagram.

When used for performance improvement, the DoView strategy/outcomes diagram is used diagnostically. Each of the boxes in the diagram is examined to see if there is a problem with achieving it. The boxes are typically marked up with traffic lights as is shown in this Using a DoView Strategy/Outcomes Diagram for Performance Improvement Tool (F1).

For boxes marked with red or orange traffic lights, performance improvement projects (PIPs) are developed to target the identified problem area. The initiative's DoView, marked up in this way, provides a clear overview of where the initiative needs to make improvements. The Performance Improvement Projects (PIPs) provide the details of exactly how the initiative will go about improving its performance.

Whether or not the entity follows through on its Performance Improvement Projects can be easily tracked following the performance review by reviewing the DoView diagram regularly and asking the initiative what progress it has made on each Performance Improvement Project.
