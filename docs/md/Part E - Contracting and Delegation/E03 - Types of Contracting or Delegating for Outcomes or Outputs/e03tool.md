# DoView Tool E3 — Types of Contracting or Delegating for Outcomes or Outputs

> **Pair:** [Question](e03question.md) · Tool (this page)

Below are the four types of contracting or delegating for outcomes or outputs. When doing any type of delegating or contracting it is always useful to discuss exactly what is being contracted for against a DoView strategy/outcomes diagram that has whether indicators are controllable and not-necessarily controllable (e.g. outcomes) marked up on it. Doing this allows parties to immediately identify the specific things that are being contracted for in the context of the 'This-Then' logic of the initiative.

## Diagram

| Type | What is contracted for |
|---|---|
| 1. Contracting for outputs only (goods and services produced by the initiative) | Outputs only |
| 2. Contracting for controllable indicators | Outputs **+** Any other controllable indicators that can be found above the outputs level |
| 3. Contracting for controllable indicators plus 'managing for outcomes' | Outputs **+** Any other controllable indicators that can be found above the outputs level **+** Proving that you are 'aligned' and optimizing your focus on outcomes, not only on outputs ('managing for outcomes')\* |
| 4. Contracting for not-necessarily controllable outcomes | Contracting for the achievement of outcomes even if not-necessarily controllable |

\* See How an Agency, Provider, Policy, Organization or Initiative Can Prove it is Managing For Outcomes, Not Only Outputs Tool (E5).
