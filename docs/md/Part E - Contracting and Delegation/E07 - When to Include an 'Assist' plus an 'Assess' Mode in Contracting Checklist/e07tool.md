# DoView Tool E7 — When to Include an 'Assist' plus an 'Assess' Mode in Contracting Checklist

> **Pair:** [Question](e07question.md) · Tool (this page)

1. Can you afford for any, or particular, suppliers/providers to fail? IF NO, THEN CONSIDER INCLUDING AN ASSIST MODE.

2. Do you need to build sector capability? IF YES, THEN CONSIDER INCLUDING AN ASSIST MODE.

3. Will sector/community morale be damaged by a provider failing? IF YES, THEN CONSIDER INCLUDING AN ASSIST MODE.

4. Is there a question of the provision of people's statutory rights being threatened by the failure of a supplier/provider? IF YES, THEN CONSIDER INCLUDING AN ASSIST MODE.

5. Is there an ongoing entity that needs to maintain its capital on an inter-generational basis (e.g. an ethnic tribal body)? IF YES, THEN CONSIDER INCLUDING AN ASSIST MODE.
