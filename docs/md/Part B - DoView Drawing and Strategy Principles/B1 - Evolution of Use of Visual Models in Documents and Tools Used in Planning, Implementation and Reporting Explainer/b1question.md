# How are the tools for planning and action evolving over time (the advent of visual drill-down strategy/outcomes diagrams)?

> **Pair:** Question (this page) · [Tool](b1tool.md)

Planning, implementation and reporting is complex, involving multiple organizations, providers, stakeholders and communities. Traditionally, various text-based and table-based formats have been used in planning documents to work with and communicate planning information. Now, however people around the world are moving to a new planning paradigm using visual models and diagrams of various types. These visual models go by names such as: logic models, theories of change, program theories, results chains, intervention logics, strategy/outcomes diagrams etc.

DoView Planning uses one standardized type of visual diagram built according to a set of rules, the DoView Drawing Rules, to make sure that they are fit for purpose for use in all stages of the planning and implementation process. DoView strategy/outcomes diagrams can be used with any type of organization, policy, strategy or initiative. Their format has been developed after developing hundreds of different diagrams for different agencies, policies, and initiatives across all sectors.

DoView strategy/outcomes diagrams can be visualized and used in standard software such as Powerpoint or Google Slides, or in DoView Planning's award-winning legacy app\*. Anyone is free to include the legacy app's unique features that assist when doing DoView Planning (with acknowledgment). Over time it is hoped that other apps, platforms and systems will provide increased support for doing specialized aspects of DoView Planning, in particular DoView Visual Alignment which allows you to see if projects/activities are aligned to priorities.

Regardless of the software you use to visualize such diagrams, DoView strategy/outcomes diagrams are at their most powerful when used in real-time in front of meetings with decision-makers, planners and stakeholders. When used in this way, participants can see a 'helicopter' overview of the outcomes being sought but also quickly 'drill-down' to see any detail they need to see and discuss.

\* This award winning software was developed by the author and has been used in 55+ countries.
