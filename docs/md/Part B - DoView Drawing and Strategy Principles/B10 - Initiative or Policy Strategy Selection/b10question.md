# Which strategies should government or nonprofits select to achieve their outcomes and include in DoView strategy/outcomes diagrams?

> **Pair:** Question (this page) · [Tool](b10tool.md)

Government agencies, nonprofits, providers, policies and initiatives use a range of different strategies to achieve their outcomes. This Initiative or Policy Strategy Selection Tool (B10) provides a comprehensive set of strategies available for use by government agencies.

Sometimes government planners sit down with a blank piece of paper and start to think up a list of strategies that could potentially be used to achieve particular outcomes. The success of this approach relies on them holding in their heads a comprehensive list of all possible strategies that they could deploy.

This tool is designed to make this task easier by providing a list of possibilities from which strategies can be selected.

First, the tool identifies a set of 'approaches'. These range from 'Educational/informational' through to 'Policy, Rules, and Standards Development'.

Second, for each approach, a set of 'methods' is identified.

Third, a set of 'styles' is identified for each approach.

Fourth, the tool provides a 'focus/setting' list from which a planner can select the focus of their strategy and the setting in which it should be implemented.

The tool can be used when building a DoView strategy/outcomes diagram for a government agency, nonprofit, provider, policy or initiative. It can also be used when reviewing such an initiative. It is used to answer the question: 'Have all of the possible strategies that could potentially be used been considered for use within the particular DoView strategy/outcomes diagram under review.
