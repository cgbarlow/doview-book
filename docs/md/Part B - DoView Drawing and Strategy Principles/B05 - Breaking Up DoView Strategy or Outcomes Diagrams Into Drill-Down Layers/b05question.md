# What is the best way of structuring DoView drill-down strategy/outcomes diagrams?

> **Pair:** Question (this page) · [Tool](b05tool.md)

The 'Archery Initiative' DoView Strategy/Outcomes Diagram Example (B4) on the previous page provided a simple example. However, real world agency, provider, policy, or initiative DoView strategy/outcomes diagrams will almost always be more complex. Such complexity presents a visualization challenge. One approach is to draw large single-page poster versions of diagrams. These will show all aspects of what an initiative is attempting, but they can be overwhelming for the reader and their size means it is not feasible to dataproject and edit them in real-time during meetings or easily use them on computers and tablets. For DoView strategy/outcomes diagrams to be practical for day-to-day use, they need to be able to be edited across all of the common communication platforms used to plan and implement government action: letter-sized paper; larger poster sized paper; electronic versions on desk-top screens, tablets and data projected versions.

If people are going to 'own' a DoView strategy/outcomes diagrams they are using, they need to be able to view and edit it in real-time during strategy meetings. The 'lowest common denominator' diagram size for such real-time editing is a data projected model on a normal-sized screen or using it on a tablet. To make it easy to edit such diagrams in real-time meetings, they are broken up into a small overview page and a set of similarly small-sized drill-down pages.* With suitable software you can get a helicopter view and then drill down for more detail.

This drill-down approach also has the advantage that, because they are smaller in size, readers can quickly assimilate the content of a single drill-down 'chunk' of the strategy/outcomes diagram faster than when looking at a large all-inclusive poster. Therefore DoView diagrams used within DoView Planning, whereever possible, should be broken up into smaller drill-down pages. These should be based on chunks of the diagram that will intuitively make sense to readers. One way you can think about these drill-down pages is that they are different 'lens' or 'perspectives' on what an agency, provider, policy, or initiative is trying to do. Of course, the individual drill-down chunks can also be reassembled into a poster version for an overview that still maintains the separate, easily assimilable, chunks.

\* You can create drill-down DoViews in PowerPoint, Google Slides or the DoView Planning legacy app.
