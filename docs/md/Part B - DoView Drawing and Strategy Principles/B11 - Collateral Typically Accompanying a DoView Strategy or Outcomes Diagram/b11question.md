# What collateral is usually produced around a DoView strategy/outcomes diagram when doing DoView Planning?

> **Pair:** Question (this page) · [Tool](b11tool.md)

There is a standard set of collateral that is usually produced when doing DoView Planning. This includes the following elements.

A DoView Booklet. This is a printed booklet that contains one or more DoView diagrams developed for the initiative, agency, provider, policy or strategy.

A clickable drill-down webpage version of the DoView diagram on an intranet or on the internet in Google Slides or the DoView Planning legacy app.

Printed poster(s) of the DoView diagram(s) that provide an overview of all of the DoView(s) in the DoView Booklet for use in meetings. Such posters are also sometimes put up permanently in a prominent place in an organization's office so that staff and visitors can quickly overview the organization's strategy and outcomes within a few minutes.

Documentation regarding how the DoView diagram(s) have been built, the DoView Drawing Rules and how they are going to be used. Relevant tools from this DoView Planning book that are being used in particular implementation of DoView Planning. This should include acknowledgment to this book and a link to DoViewPlanning.Org where people can get more information.

An electronic version of the DoView strategy/outcomes diagrams in PowerPoint, Google Slides or the DoView Planning legacy app that can be used in real-time in meetings to amend the DoView diagram(s) and to use tools from DoView Planning such as DoView Visual Alignment.

Other collateral can be developed. For instance, images or live links to subpages from the DoView can be linked to directly to any online Dashboards created to report on indicators measuring boxes from the relevant DoView diagram subpage.
