# What makes a good indicator for an organization, agency, provider, policy or initiative?

> **Pair:** Question (this page) · [Tool](d06tool.md)

There are many indicators that can potentially be measured relating to the work of organizations, agencies, providers, policies, strategy and initiatives. This Good Indicator Checklist (D6) helps you analyze, accept, reject or improve any indicator that you are considering measuring.

It complements the Putting Indicators Onto A DoView Strategy/Outcomes Diagram Tool (D5), which shows the position of indicators within the overall strategy/outcomes diagram for an agency, provider, policy, strategy or initiative.

The list of features of a good indicator used in this tool is drawn from the Features of Elements That can be Included in a DoView Strategy/Outcomes Diagram or Outcomes Framework (B8).
