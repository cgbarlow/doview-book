# DoView Tool D2 — Strategy and Outcomes System Terminology

> **Pair:** [Question](d02question.md) · Tool (this page)

The component from the DoView Planning Framework (Strategy and Outcomes Components (Building Blocks) Diagram) (D1) being referred to by terms used in strategy, planning, outcomes, monitoring, performance management, alignment, indicators, deliverables, evaluation, accountability and value for money work.

## Strategy and planning

| Component | Equivalent terms |
|---|---|
| **A — Evidence-based STRATEGY specified** | Initiative planning, intervention logic, logic modeling, theory of change, program theory, strategy mapping, results chain, outcomes model, strategy model, visual strategy diagram. |
| **B — PRIORITIZATION of outcomes** | Priority setting, goal setting, key driver analysis. |
| **C — ALIGNMENT of activities** | Line-of-sight alignment, resource alignment, strategic focusing, enterprise portfolio management, investment mapping, budget allocation. |

## Monitoring performance & outcomes

| Component | Equivalent terms |
|---|---|
| **D — OUTPUTS and indicators the initiative controls** | Lower-level indicators, monitoring, performance measurement, outputs, key performance indicators, KPIs, deliverables, controllable indicators, attributable indicators. |
| **E — Higher-level OUTCOMES that are not-necessarily controllable** | High-level indicators, outcomes, state indicators, trends tracking, monitoring, not-necessarily controllable indicators, not-necessarily attributable indicators, community indicators, sector indicators. |

## Evaluation

| Component | Equivalent terms |
|---|---|
| **F — IMPLEMENTATION evaluation for optimization** | Developmental, formative, implementation evaluation, quality management, performance improvement, organizational or initiative review, program fidelity, action research. |
| **G — PROCESS evaluation for description** | Process evaluation, context-setting, program documentation. |
| **H — IMPACT evaluation to prove impact** | Impact evaluation, outcomes evaluation. |
| **I — ECONOMIC and SUMMATIVE evaluation** | Initiative costing, cost-effectiveness analysis, cost-benefit analysis, value for money. |
