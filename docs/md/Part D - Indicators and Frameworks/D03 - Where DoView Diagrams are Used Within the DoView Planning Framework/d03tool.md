# DoView Tool D3 — Where DoView Diagrams are Used Within the DoView Planning Framework

> **Pair:** [Question](d03question.md) · Tool (this page)

Below shows how DoView Diagrams are used in each part of the DoView Planning Framework.

## Strategy and planning

| Component | How DoView diagrams are used |
|---|---|
| **A — Evidence-based STRATEGY specified** | Captured in a DoView strategy/outcomes diagram |
| **B — PRIORITIZATION of outcomes** | Marked up onto a DoView |
| **C — ALIGNMENT of activities** | Done with DoView Visual Alignment |

## Monitoring performance & outcomes

| Component | How DoView diagrams are used |
|---|---|
| **D — OUTPUTS and indicators the initiative controls** | Put onto a DoView next to boxes they measure and marked up as controllables |
| **E — High-level OUTCOMES that are not-necessarily controllable or attributable to the initiative** | Put onto a DoView next to boxes they measure |

## Evaluation

| Component | How DoView diagrams are used |
|---|---|
| **F — IMPLEMENTATION evaluation for optimization** | Questions put on DoView |
| **G — PROCESS evaluation for description** | Questions put on DoView |
| **H — IMPACT evaluation to prove impact** | Questions put on DoView |
| **I — ECONOMIC and SUMMATIVE evaluation** | Questions put on DoView |
