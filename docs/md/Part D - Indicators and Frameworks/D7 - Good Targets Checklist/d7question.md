# How is it best to set targets for an organization, provider, policy, strategy or initiative?

> **Pair:** Question (this page) · [Tool](d7tool.md)

Specific levels (called 'targets' or sometimes 'goals') to be achieved are often set on the indicators for an organization, provider, policy, strategy or initiative. Target(s) are formally defined as levels set on the indicators (measures) of boxes within a DoView strategy/outcomes diagram.

Sometimes when targets are presented, some of the implicit underlying DoView strategy/outcomes diagram that lies behind their selection is articulated. For instance, in a block of text about the 'rationale' why the target is relevant to the initiative. In DoView Planning suggested this is information is best seen as part of the 'This-Then' logic which is best set out in the DoView strategy diagram.

There are four considerations when setting targets. First is the question of 'coverage'. A set of targets should be comprehensive in the sense of covering a sufficient range of the priority steps or outcomes being pursued. If you always show targets against the relevant DoView strategy diagram boxes, you can see exactly which boxes the targets are related to and make sure that they are not just focused on a limited number of areas in the DoView strategy diagram.

Second, there is the 'level' within the relevant DoView diagram of the indicators on which targets has been set. Even if targets cover a comprehensive set of outcome areas as discussed above, some may be at very low-levels within some outcome areas. As discussed in the Proxy Indicator Analysis Tool (D11), the position of an indicator (high or low) can be tested by putting the indicator onto the relevant DoView strategy/outcomes diagram.

Third, there is the actual level the target has been set at on the particular indicator. This is a judgment call as the target needs to be somewhat aspirational while not being grossly unrealistic.

Fourth, targets can be set as targets for an individual organization, agency, provider, policy initiative, or more broadly. Targets are set as 'sector' or 'community-level' targets rather than individual initiative targets when not-necessarily controllable by an individual organization, agency, provider, policy, or initiative.
