# How can you build a DoView Monitoring and Evaluation (M&E) Plan?

> **Pair:** Question (this page) · [Tool](g25tool.md)

Currently, monitoring and evaluation (M&E) plans appear in various formats. The simple fact that a well-presented evaluation plan written in technical language has been prepared for an initiative often lends credibility to the idea that the evaluation questions and methodology are suitable. However, for those without extensive knowledge of evaluation methodology, it is challenging to determine whether a particular evaluation plan is truly appropriate. This problem is worsened by the lack of a standardized format for an evaluation plan, and this will become more problematic as AI lets anyone produce a convincing technical evaluation plan in minutes. Despite these difficulties, it is essential that purchasers, funders, suppliers, providers, and stakeholders can fully understand what is being proposed in a monitoring and evaluation plan and whether it is reasonable.

DoView Planning addresses this issue by suggesting that evaluation plans be set out in a standard format, based on a visual DoView strategy/outcomes diagram. Regardless of their evaluation background, once people become familiar with the standard DoView Monitoring and Evaluation Plan format, they can quickly find the key information they need. In addition, DoView M&E Plans include tools that clarify many of the core issues in evaluation planning.

Within a DoView M&E Plan, mapping indicators onto the relevant DoView diagram (D5) clarifies how much attribution can be achieved just by measuring controllable indicators. The Mapping a Set of Evaluation Questions Onto a Strategy/Outcomes Diagram Tool (G7) means people can see the level at which evaluation questions are being struck, the type of evaluation being done, and whether duplicate questions exist. The DoView Impact Evaluation Suitability Check (G10) helps make sure all major impact evaluation design types have been considered and clarifies the rationale for those selected. This tool also supports analysis of the appropriateness, feasibility, affordability and credibility of options. The Types of Cost-Benefit and Value for Money Analysis Tool (G15) checks that the right kind of economic evaluation is used, given the effect sizes available from impact evaluation. As a result, the DoView Monitoring and Evaluation (M&E) Plan (G22) provides a standardized format that helps even non-experts assess the quality of an M&E plan.
