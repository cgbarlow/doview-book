# How can you assess the level at which a set of evaluation questions is struck, its comprehensiveness and if there are any duplicate questions?

> **Pair:** Question (this page) · [Tool](g07tool.md)

The Generic Evaluation Question List (G6) showed how to identify generic evaluation questions for any evaluation of any organization, agency, provider, policy or initiative. This Mapping a Set of Evaluation Questions Onto a DoView Strategy/Outcomes Diagram Tool (G7) shows to how analyze an already existing proposed set of evaluation questions. Map each proposed evaluation question onto the particular initiative's DoView strategy/outcomes diagram. Then check them off against the Generic Evaluation Questions List (G6) to make sure all of the generic evaluation question areas have been included.

The purpose of mapping evaluation questions in this way is to identify duplicate evaluation questions. You can find them because duplicate questions with the same meaning but different wording will be placed in a position on the DoView strategy/outcomes diagram. For example, in this tool, you can see there are two duplicate impact evaluation questions: "Evaluation Q 4 – Did the Archery Agency lead to more arrows hitting the bulls-eye?" and "Evaluation Q 4a – Did the Archery Agency achieve its outcome?" Second, mapping evaluation questions onto an initiative's DoView diagram lets you see the "level" at which any evaluation question is being struck. This is important because, as we can see from Defining an Initiative's Outcomes (G15), what is regarded as the "outcomes" of an initiative can differ depending on how one defines an "outcome." This then affects the outcome level at which an impact evaluation question is being struck. It is important for evaluation planners and commissioners to see if the "outcomes level" is being moved down an initiative's DoView strategy/outcomes diagram by changing the way an evaluation question is worded. Once an evaluation question gets too low down a strategy/outcomes diagram, it is not a high-level impact evaluation question anymore.

This Evaluation Question Mapping Tool (G7) also allows evaluation planners and others to clearly differentiate between implementation and impact evaluation questions. Implementation evaluation questions are to the left of the strategy diagram, and impact evaluation questions are to the right.
