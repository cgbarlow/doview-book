# DoView Tool G20 — Evaluation Data Collection Types

> **Pair:** [Question](g20question.md) · Tool (this page)

## Data Collection Types and Sub-types

| Data collection type | Sub-types |
|---|---|
| Directly observing | Participant; Non-participant |
| Asking others | Interviews; Questionnaires |
| Dialogue facilitation | Focus Groups; Stakeholder discussions & consultation; Public consultation; Dialogue processes (e.g. Rich Dialogue Processes (RDP)) |
| Document analysis | Published; Grey literature; Administrative |
| Indicators | New indicators set up; Indicators already collected |
