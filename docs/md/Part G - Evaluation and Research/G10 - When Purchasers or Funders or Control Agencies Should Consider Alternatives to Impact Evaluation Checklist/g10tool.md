# DoView Tool G10 — When Purchasers/Funders or Control Agencies Should Consider Alternatives to Impact Evaluation Checklist

> **Pair:** [Question](g10question.md) · Tool (this page)

Sometimes purchasers/funders or central agencies issue a blanket demand that suppliers/providers, organizations and initiatives 'do impact evaluation'. However, not all initiatives merit impact evaluation. This checklist shows some of the instances in which impact evaluation is inappropriate. For each question below, the appropriate type of evaluation is shown that is identified in the DoView Planning Framework (D1). Where there is neither a 'Y' nor an 'N' you may or may not decide to do the particular type of evaluation. This approach can be used in what is called 'evaluability assessment' and the Strategic Evaluation Approach (G2).

## Diagram

| Question (IF YES) | Initiative optimization (Implementation Evaluation) | Initiative course and context description (Process Evaluation) | Impact Evaluation | Economic and Summative Evaluation |
|---|---|---|---|---|
| 1. Has the same type of initiative with similar people / units in similar settings been shown to work? | Y | | N | |
| 2. Is the intervention not yet in a stable form or not enough time has passed for outcomes to occur? | Y | Y | N | |
| 3. Are there limited resources for the evaluation? IF YES, IT MIGHT BE BETTER TO MAXIMIZE THE CHANCES OF THE INITIATIVE SUCCEEDING BY USING IMPLEMENTATION EVALUATION | Y | | N | |
| 4. Is the initiative likely to work but too expensive to be rolled out more widely? IF YES, USE PROCESS EVALUATION TO WORK OUT WHY A STRATEGIC EVALUATION APPROACH IS NOT BEING USED (G2) | | Y | N | Y |
| 5. Is the initiative not a priority in terms of the type of initiatives that need to be done in the sector? IF YES, DO SAME AS IN 4. | | Y | N | |

Where there is neither a 'Y' nor an 'N' you may or may not decide to do the particular type of evaluation.
