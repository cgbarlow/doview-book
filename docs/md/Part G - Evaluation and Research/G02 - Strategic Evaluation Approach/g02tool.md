# DoView Tool G2 — Strategic Evaluation Approach

> **Pair:** [Question](g02question.md) · Tool (this page)

Below are the steps that are undertaken when using a strategic evaluation approach. In order to do these steps draw on relevant DoView Planning tools. As AI develops it should be able to assist with many of the tasks below.

1. As much as possible, attempt to have evaluation resources pooled rather than exclusively allocated to evaluating a particular initiative. Ideally, evaluation resources should be seen as one single Research and Development (R&D) funding pool informing ongoing strategic decision-making for the sector or setting.
2. Develop a DoView strategy/outcomes diagram for the setting or sector in which the initiative to be evaluated is located.
3. Ensure that previous evaluation findings relevant to the issue have been captured in the 'This-Then' claims being made in the DoView diagram.
4. Have authoritative stakeholder/decision-maker/subject experts input into identifying the strategic evaluation questions that need to be answered to make progress on achieving the outcomes within the setting or sector DoView.
5. Set up a permanent high-level Strategic Evaluation Oversight Group that meets periodically to decide which strategic evaluation questions should be answered next, which specific initiatives should be evaluated to answer which evaluation questions and develops indicative estimates for how much it will cost to answer each strategic evaluation question so it can decide on the best way to allocate evaluation resources. It would then do 6-12 below.
6. Put the evaluation questions onto the DoView diagram next to the boxes they relate to.
7. Prioritize which evaluation question should be focused on in the next period.
8. Work out what type of evaluation of the current initiative will help answer the priority evaluation questions.
9. Do not prejudge the level of evaluation resources that should be spent on a particular initiative. It will depend on the extent to which evaluating that initiative will answer particular strategic evaluation questions using selected evaluation types.
10. Map onto the DoView diagram which current evaluation projects are answering which strategic evaluation questions.
11. Map the results of evaluation projects back onto the sector or setting DoView.
12. Use the evaluation results that have been mapped onto the DoView in the next iteration of the DoView Planning cycle as the Strategic Evaluation Oversight Groups determines which initiatives are priorities for funding in the next period.
