# What are the impact evaluation design types you need to consider when planning an evaluation?

> **Pair:** Question (this page) · [Tool](g11tool.md)

Once you have decided that doing an impact evaluation is appropriate (see When Funders and Central Agencies Should Consider Alternatives to Impact Evaluation Checklist (G10)), the next step is for evaluation planners to sit down and try to work out which type of impact evaluation to do. Often this problem is approached as if one is starting from scratch and attempting to think up a suitable impact evaluation design for what they are evaluating.

Doing it this way relies on evaluation planners (and others such as evaluation commissioners) holding in their heads a detailed list of all of the possible impact evaluation designs that can be used. If people do not have an exhaustive knowledge of impact evaluation design types, then they may forget to consider one, or more, potentially useful impact evaluation design types.

This Impact Evaluation Design Types Tool (G11) helps anyone identify which impact evaluation design type(s) could be used in a particular impact evaluation. This is a more comprehensive, efficient, and less stressful way of working out what type of impact evaluation design types to use for a particular agency, provider, project or initiative.

Also, when evaluation commissioners or others are looking at proposals evaluators have submitted, such proposals describe the impact evaluation designs they are proposing using somewhat different terminology. Working out the exact design type being offered can be time-consuming even if the evaluation commissioner is being assisted by an independent evaluation methodology specialist. This is even more difficult for people such as some managers and policy analysts with less specific impact evaluation technical expertise, who often find themselves involved in commissioning evaluations.

This Impact Evaluation Design Types Tool (D11) provides a standard terminology for describing impact evaluation design types and can be used in evaluation planning, commissioning, and reporting. This tool can be included in evaluation Requests for Proposals RFPs. Then different evaluators submitting proposals can use the same terms to describe the evaluation designs in their proposals, making comparison easier.
