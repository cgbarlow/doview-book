# DoView Tool I4 — How to Set up a Room For Interactive DoView Strategy/Outcomes Diagram Work Checklist

> **Pair:** [Question](i4question.md) · Tool (this page)

At the heart of DoView Planning is interacting with interactive drill-down DoView strategy/outcomes diagrams. This is easy when people are using appropriate apps or platfomrs on their own computers or screen sharing in virtual meetings, but for face to face meetings meeting rooms need to be optimized facilitate real-time interaction with DoViews as set out in this checklist.

1. Will everyone in the room be able to see a sufficiently large version of the DoView diagram to read the smallest font on the diagram and work with it all day without becoming fatigued? IF THE SCREEN IS TOO SMALL, CONSIDER DATA PROJECTING ONTO A LARGER WHITE WALL. IF NOT POSSIBLE, SUPPLEMENT WITH SCREEN SHARING ON PEOPLE'S TABLETS OR COMPUTERS.

2. Is there excessive glare in the room that will make interacting with the DoView diagram difficult? IF YES, CAN CURTAINS BE PULLED, LIGHTS TURNED DOWN?

3. Is there space for two different DoView diagrams or different subpages to be projected at the same time? IF NOT, THE SECOND DOVIEW'S PAGES CAN BE CIRCULATED AS PRINTED VERSIONS, OR READ ON TABLETS OR COMPUTERS.

4. Is there comfortable ergonomic table space for whoever is working with the DoView diagram to be able to edit it in real-time as the group works? IF NOT, SET UP.

5. Is there sufficient wall space for poster versions of relevant DoView diagrams to be put up so that people can look at them and discuss them informally during meeting breaks? PUT DOVIEWS ON WALLS.

6. Are printing and photocopying facilities at hand so that draft pages of the DoView diagram can be rapidly printed out and made available to the participants in the meeting? HAVE PRINTING FACILITIES AVAILABLE.
