# How can the use of an Investment Logic Map (ILM) be enhanced with a DoView strategy/outcomes diagram?

> **Pair:** Question (this page) · [Tool](h5tool.md)

Investment Logic Mapping (ILM) is a process where what an initiative is trying to do is set out in the following sequence:

Problems
Benefits
Strategic Responses
Solutions

How does this way of setting out what is being sought by an initiative be enhanced by the use of a DoView strategy/outcomes diagram?

A DoView strategy/outcomes diagram can be seen as allowing you to spell out in much more detail the logic between two of the headings used within Investment Logic Mapping. These are the 'problems' and 'benefits' headings. A DoView used with an ILM in this way can tell a much more detailed 'This-Then' story of how it is believed the initiative will achieve its outcomes. It can do this because it is able to include as many drill-down subpages as are necessary to fully describe the initiative's logic. It allows you to detail what is stated at a high level in the 'strategic responses' and 'solutions' columns of the ILM.

The Complementing an Investment Logic Map (ILM) With a DoView Strategy/Outcome Explainer (H5) shows how a DoView can be used when using an ILM.

Using a DoView diagram to enhance an ILM allows you to elaborate in more detail on the 'This-Then' logic of what is envisioned for an initiative. If you go on to use DoView Planning for an initiative, then the same DoView can be used in all stages of planning, implementation and measuring its impact. The use of a DoView diagram ensures that you can be more certain of achieving tight alignment between what was initially articulated in an ILM at the time of seeking funding and what actually happens when a funded initiative is implemented on the ground.
