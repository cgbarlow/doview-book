# DoView Tool J3 — Using AI to Speed Up DoView Planning

> **Pair:** [Question](j3question.md) · Tool (this page)

DoView Planning is being supercharged due to the introduction of AI. Prior to this, there were a number of bottlenecks when doing DoView Planning. These included: having to have experts draw fit-for-purpose DoView diagrams, identifying indicators, and doing DoView Visual Alignment, where you map projects/activities onto the steps/outcomes in a DoView. We are developing the following DoView AI prompts to overcome these bottlenecks. These work on ChatGPT's 5 (Thinking) model. Obtain access by getting your own ChatGPT Plus subscription. Check DoViewPlanning.Org for our most up-to-date AI prompts. Here are some prompts we are currently working on. Note that when using AI for DoView Planning or any purpose, you should always carefully check what it comes up with.

- Creating PowerPoint an organization or initiative's DoView strategy/diagrams from either information you provide or by AI looking up information about it on the internet (use our free AI prompt available at DoViewPlanning.Org now).
- Adding traffic light and priority boxes to existing PowerPoint DoViews, so you can visually indicate progress and set priorities (A, B, C, D, E, or BAU).
- The user providing source material that the AI can use to create a 'traffic-lighted' DoView showing the current achievement level of boxes within the DoView and providing a justification for how the AI has traffic-lighted each box.
- Having AI go through a DoView and identify possible indicators that can be used to measure the boxes in it.
- Identifying possible evaluation questions and locating them next to the relevant boxes within the DoView.
- Developing a DoView Monitoring and Evaluation Plan for any DoView.
- Providing a first cut showing visual alignment between projects/activities and the steps/outcomes within a DoView when undertaking DoView Visual Alignment.
- Experimenting with getting AI to overview the progress that is currently being made with implementing an organization or initiative's DoView diagram and providing draft suggestions as to which boxes in a DoView should be priorities for action in the next planning period.
